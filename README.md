# NumBomb
## 一个基于Unity2d开发的像素画风线下聚会小游戏
### 游戏流程
#### 运行程序后进入主菜单，主菜单数字可以设置游戏关键数字的生成范围，默认范围为0-100。
#### 设置范围后开始游戏，通过数字按钮键入猜测的数字，根据输入的数字推进游戏流程：没猜中则缩小范围，猜中则游戏结束。
#### 功能实现：长按退格(backplace)暂停游戏，呼出暂停菜单。暂停时点击回车(enter)继续游戏。
